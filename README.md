### 4. 코드블록
**인라인 코드**  
`System.out.println("Hello, world!");`

**코드 블록**
```java
public class Hello {

	public static void main(String []args){
		//console : "Hello, world!" 메시지 출력
		System.out.println("Hello, world!");
	}
}
```
